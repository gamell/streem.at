# streem.at API

## Installation

### Install homebrew

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

### Install node & npm

```
brew install node
npm install
```

###Install and configure mongoDB

```
brew install mongod
sudo mkdir /data/db
sudo chmod 777 /data/db
```

or

```
brew upgrade mongodb
sudo mkdir /data/db
sudo chmod 777 /data/db
```

### Other Tools

#### eslint

```
npm install -g eslint
```
